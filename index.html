Find the best time to shoot

Glance at Rizon’s 24-hour clock to get an idea of when the light is going to be at it’s best. Rotate it to find out when each segment of light will occur. See up to a week ahead to plan when to shoot.

Never forget to get outside

Set Automatic Alerts for weekdays or weekends, and mornings or evenings to be reminded when to grab your camera. Fine tune the threshold of when to be reminded so you’re always ready.

Golden Hour across the globe

Planning a trip and want to know when Golden Hour will happen? Add a custom location and have access to that location offline. Great for when you have no data abroad.

Additional features

Coming soon!

We’re working on some additional features for Rizon, such as the ability to set custom “one-off” reminders for a chosen location and date, increased date range, along with a Today Widget, and possibly an Apple Watch app.
